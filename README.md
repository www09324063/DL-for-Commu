## 高铁研究
### 协同感知
在自动驾驶与轨道交通智能化领域，协同感知与动态控制是当前的热点方向。以下是 alphaXiv 为你筛选的几篇相关研究论文：
1. **Vehicle-to-Everything Cooperative Perception for Autonomous Driving** (2年前发表)这篇综述详细介绍了车联网（V2X）协同感知的基本架构和关键技术。虽然侧重于道路车辆，但其关于多传感器数据融合与通信延迟对感知影响的分析，与本文中列车协同感知的理论基础高度相关。
[链接](https://alphaxiv.org/abs/2310.03525)
2. **UniMM-V2X: MoE-Enhanced Multi-Level Fusion for End-to-End Cooperative Autonomous Driving** (2个月前发表)该论文探讨了利用专家混合模型（MoE）进行多层级特征融合，以解决协同驾驶中的感知局限。其提出的端到端协同框架对于理解未来列车自主运行系统中的复杂决策过程具有参考价值。
[链接](https://alphaxiv.org/abs/2511.09013)
3. **Deep Reinforcement Learning in Autonomous Car Path Planning and Control: A Survey** (2年前发表)该综述涵盖了深度强化学习在自主导航和控制中的应用。在列车跟踪间隔的优化策略中，强化学习常被用于在动态变化的环境中寻找最优的制动与加速曲线，以实现效率与安全的平衡。
[链接](https://alphaxiv.org/abs/2404.00340)
