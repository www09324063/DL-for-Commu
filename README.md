# DL-for-Commu
## 算法高级工程师
1. 电梯智能控制算法研发

- 设计并实现基于强化学习（RL） 与实时运筹优化的电梯群控调度算法，优化多目标函数（等待时间、能耗、运载效率）

- 构建动态客流预测模型，融合楼宇IoT传感器、历史OD矩阵等异构数据，实现分钟级调度策略自适应

- 研发边缘端轻量化模型，解决电梯场景下低延迟（<100ms）与高可靠性的工程挑战

- 搭建数字孪生仿真平台，对调度策略进行A/B测试与压力验证

2. 全渠道销量预测算法研发

- 主导构建覆盖线上线下、多SKU层级的产品销量预测模型，支持从小时级到季度级不同时间粒度

- 设计时空图神经网络（ST-GNN） 与Transformer混合架构，挖掘区域、渠道、促销等复杂特征关联

- 开发模型可解释性工具链，量化分析价格、营销、季节性等因素对销量的因果影响

- 建立自动化模型迭代管线，实现特征工程、超参搜索、模型部署的MLOps闭环
